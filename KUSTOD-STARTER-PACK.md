# KUSTOD Starter Pack — v0.1
### Turn any AI tool into a consistent working partner. 10-minute setup.

> Stop re-explaining yourself to AI every day.
> A KUSTOD is a working partner configured around your role, your style, your memory habits — and one rule most AI quietly breaks: **it pushes back when you're wrong.**

**Works in:** ChatGPT · Claude · Copilot · Gemini — any chat AI. No install. No login.
**Time to first setup:** ~10 minutes. **Time to first useful result:** same day.

---

## What is a KUSTOD?

Not an agent. Not a chatbot. A **working partner** you configure once around four things:

1. **Your role & goals** — so it stops giving generic answers
2. **Your working style** — concise vs deep, direct vs gentle
3. **A memory habit** — so you don't re-explain yourself every session
4. **A correction loop** — so it challenges weak ideas instead of flattering you

The value isn't magic. It's **consistency over time** — a partner that remembers how to work *with you*, and tells you the truth.

**Why "KUSTOD" and not "agent"?** Because "agent" promises autonomy nobody actually ships. A KUSTOD is honest about what it is: a configured partner, with you in the loop.

## The 10-Minute Activation

Fill in the scaffold, paste it as the first message of a fresh chat, and you have a working KUSTOD. Re-paste any time you start a new thread (until your tool's memory holds it).

### Block 1 — Who are you?
```
ROLE:            (e.g. B2B sales, SOC analyst, parent, founder)
GOALS:           (the 1–3 things that actually matter this month)
RECURRING WORK:  (what you do over and over — meetings, reports, follow-ups)
COMMUNICATION:   (how you like answers — short/long, blunt/soft, bullets/prose)
CONSTRAINTS:     (time, tools, what you can't share, hard limits)
HOW TO CHALLENGE ME: (what you want it to push back on)
```

### Block 2 — Name it
Give your KUSTOD a working name. Not for cuteness — a name makes it **easier to refer to, recall, and improve over time.**

### Block 3 — Working style
Pick one from each pair: `concise / deep` · `direct / gentle` · `skeptical / creative` · `structured / conversational`

### Block 4 — Correction loop  *(mandatory — this is the whole point)*
```
- Challenge weak assumptions, don't just agree.
- Flag when you're uncertain instead of guessing confidently.
- Do NOT agree with me just because I sound sure.
- If I'm about to make a mistake, say so before I make it.
```

### Block 5 — Memory ritual
At the end of a session (or once a day/week), ask for: `what happened / what changed / what to REMEMBER / what to FORGET / next actions`. Save it. Paste it back next session. **This is the habit that replaces re-explaining yourself.** Don't assume the tool's built-in memory is enough.

### Block 6 — Packet mode (optional, powerful)
Teach it to answer important things tight: `SUMMARY · RISKS · NEXT STEP · OWNER · CONFIDENCE`

### Block 7 — Voice note
If you talk to it by voice: **voice is for flow, not for commitments.** Review anything that matters in text before you act.

## The 7-Day Test

```
Day 1  Activate. Get ONE useful thing out of it.
Day 2  Use it for real. Trigger the correction loop on purpose.
Day 3  Do a memory wrap. Paste it back tomorrow.
Day 4  Notice: did you re-explain less than usual?
Day 5  Push it — give it something ambiguous.
Day 6  Share it with one person.
Day 7  Review: what did it get WRONG? Fix the setup.
```

## Guards (the difference between a partner and a sycophant)

- 🧯 **A KUSTOD is not a person.** It's a configured working-partner pattern. Don't outsource judgment.
- 🧯 **Make the memory habit explicit.** Don't trust built-in memory to hold what matters. Save your wraps.
- 🧯 **The correction loop is mandatory.** Remove it and you've got flattery with a name. The pushback *is* the product.
- 🧯 **Protect your data.** Don't paste sensitive personal, client, or company data into general AI tools. Strip names, numbers, and secrets first.
- 🧯 **Measure artifacts, not feelings.** "It felt smart" pays no rent. "It saved me an hour" does.
