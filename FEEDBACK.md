# Tried a KUSTOD? Tell me.

v0.1 is hand-distributed on purpose — I want conversations, not anonymous downloads.

If you set one up, send me these four things (open an issue, or DM me "KUSTOD"):

1. **KUSTOD name** — what did you call it?
2. **First use case** — what did you point it at?
3. **First artifact** — what did it actually produce?
4. **One thing that confused you** — where did the pack fall short?

That's data, not applause. It's how v0.2 gets better.
